# System Diagnostic Tool

System Diagnostic Tool is a Python project for basic system and network diagnostics used in IT support and infrastructure troubleshooting.

## Purpose

This project provides a clean foundation for collecting machine status information, checking connectivity, and organizing diagnostic results in a simple and practical structure.

## Use Cases

- IT support triage for workstation and server issues
- Basic system diagnostics during troubleshooting
- Network connectivity checks for common infrastructure problems
- Structured reporting for technical support workflows
